HTML Tutorial

HTML stands for HyperText Markup Language, and it is used to create web pages. Here are some basic HTML tags you can use:

  • <html> - Defines an HTML document
  • <head> - Contains metadata and links to external files
  • <title> - Defines the title of the document
  • <body> - Contains the visible content of the document
  • <h1> to <h6> - Defines headings of different levels
  • <p> - Defines a paragraph
  • <a> - Defines a hyperlink
  • <img> - Defines an image
  • <ul> and <li> - Defines an unordered list
  • <ol> and <li> - Defines an ordered list
  • <table>, <tr>, <th>, and <td> - Defines a table
  • <form>, <input>, and <button> - Defines a form and its elements

CSS Tutorial

CSS stands for Cascading Style Sheets, and it is used to style web pages. Here are some basic CSS properties you can use:

  • color - Defines the text color
  • background-color - Defines the background color
  • font-family - Defines the font family
  • font-size - Defines the font size
  • text-align - Defines the text alignment
  • padding - Defines the padding around an element
  • margin - Defines the margin around an element
  • border - Defines the border of an element
  • display - Defines how an element is displayed
  • float - Defines how an element is positioned
